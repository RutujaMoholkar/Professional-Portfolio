Colmar Academy Capstone Project through Codecademy
The purpose of the project was to develop a single page site for an educational institution. I built a responsive site from scratch. I started from the basics of HTML, its elements and structure. Then I moved on to CSS and learned about flexbox and responsive layout and GitHub Pages. I was provided with wireframes, icons and images. I translated their design into HTML and CSS and added my own customizations, with respect to color and animations to create an attractive and functional responsive design. Colmar is an actual city in the northeast of France. I did some research and found that it is a very vibrant and colorful city. I used images of the city as inspiration for my color choices in my design.

Technologies: HTML, CSS, Flex Box, Media Queries
